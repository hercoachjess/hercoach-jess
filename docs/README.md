# docs/

Anything Jess wants to keep alongside the codebase lives here — prompts,
brand assets, coaching IP, reference material, draft lead magnets.

The repo is private to Jess's GitHub account, so coaching IP and
prompts are safe here. **Do not commit anything tied to a specific
real client** (names, photos, special-category notes) — UK GDPR
treats source control as a data store. Use first names or anonymise.

## Folders

| Folder | What goes here | Example |
| --- | --- | --- |
| `prompts/` | Reusable AI prompts Jess has refined in Claude / ChatGPT | "Meal plan voice prompt", "Check-in summary template" |
| `brand/` | Visual / verbal brand reference | Logo brief, palette hex codes, tone-of-voice rules |
| `coaching/` | Clinical reference + protocols Jess wants the AI to follow | Macro split rules, training progression principles, sources |
| `guides/` | Lead-magnet content and drafts | "Cut the Food Noise" source, future freebies |
| `notes/` | Anything else that doesn't fit above | Half-formed ideas, links, screenshots with annotations |

## How to use

- Drop files in this chat — Claude saves them into the right folder and commits to GitHub.
- Future Claude Code sessions clone the repo fresh, so they automatically see everything here.
- Read on your phone: github.com/hercoachjess/hercoach-jess → `docs/` → folder.
- ChatGPT can't read GitHub directly, but you can copy/paste from your phone in seconds.
